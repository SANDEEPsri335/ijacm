# International Journal of Advanced Computing and Mechanical Systems (IJACM)

## 🌐 Website Information

This is the official website for the **International Journal of Advanced Computing and Mechanical Systems (IJACM)**, a peer-reviewed academic journal dedicated to publishing high-quality research in the fields of advanced computing technologies and mechanical systems engineering.

## 📋 Project Details

### About the Journal
IJACM is an open-access, peer-reviewed journal that bridges the gap between advanced computing methodologies and mechanical engineering applications. The journal publishes original research articles, review papers, case studies, and technical notes.

### Website Features
- **Responsive Design**: Optimized for all devices (mobile, tablet, desktop)
- **Modern UI/UX**: Professional academic interface with intuitive navigation
- **Dynamic Content**: JavaScript-powered article management system
- **Editorial Board Portal**: Interactive profiles with LinkedIn integration
- **Multi-page Architecture**: Separate sections for all journal functions
- **SEO Optimized**: Proper meta tags and semantic HTML structure

## 🎨 Design Philosophy

### Color Scheme
- **Primary**: Deep Blue (#1a237e) - Represents academic depth
- **Secondary**: Sky Blue (#00b0ff) - Symbolizes innovation

## 🔄 Maintenance & Updates

### Adding New Content
1. **New Editors**: Add profile to `editorial-board.html` with appropriate structure
2. **New Articles**: Update article data in JavaScript arrays
3. **New Pages**: Follow existing HTML/CSS patterns

### Style Customization
Modify CSS variables in the `:root` selector:
```css
:root {
  --primary: #your-color;
  --secondary: #your-color;
  /* Update other variables as needed */
}
```

## 🌐 Deployment

### GitHub Pages
This website can be easily deployed using GitHub Pages:
1. Push code to `main` branch
2. Enable GitHub Pages in repository settings
3. Select the root folder
4. Access via `https://[username].github.io/ijamc`

### Custom Domain
To use a custom domain:
1. Update CNAME file with domain name
2. Configure DNS settings with your provider
3. Update base URL in configuration
